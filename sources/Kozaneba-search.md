# Kozaneba サーベイ（116件）

nishio自作のデジタル文房具。Movidea → Kozaneba にリブランド（2021-08起動）、現在進行形で開発継続。

公式説明: 「Kozaneba is a digital stationary to organize your thought.」

## 位置付け（最重要）

[Keichobotは言語化しKozanebaは一次元化する.md](../raw/external_brain_in_markdown/pages/Keichobotは言語化しKozanebaは一次元化する.md) に明示：

- **Keichobot** = 言語化（モヤモヤ→シンボル→関係を質問で引き出す）
- **Kozaneba** = **一次元化**（関係を客体化→ストーリーへ並べる）

→ 知的生産プロセスを **2フェーズに分業して実装** している。Kozaneba は「**既に言語化されたものを構造化・ストーリー化する**」段階を担う。

## KJ法との関係（系譜の最新形）

- 物理的なKJ法（1960s）の付箋を **線でつなぐ** ことを可能にした
- KJ法では「近接配置」で関係を表現していた → Kozaneba では「線」で明示
- → **既存の構造を保ったまま別の構造化** が可能（KJ法ではできなかった）
- 詳細: [Keichobotは言語化しKozanebaは一次元化する.md](../raw/external_brain_in_markdown/pages/Keichobotは言語化しKozanebaは一次元化する.md) "なぜ線が必要不可欠か"

## 観察

### 「一次元化」の意味

ストーリー = 一次元の線。脳内のネットワーク（多次元）を一次元の物語に並べることが知的生産の最終段階の一つ。Kozanebaの「線を引く機能」と「並べる機能」がそれを支援する。

### 5年継続の自作ツール（2021-2026）

- 2021-08起動、現在まで5年継続開発
- 開発日記の時系列が **2021年集中 → 2022-2023断続 → 2025年再活発化** という波形
- 「Kozanebaの次の進む方向についてわかると良い」 — 進路の停滞TODO（[[../wiki/concepts/曖昧な停滞.md|曖昧な停滞]] 事例）
