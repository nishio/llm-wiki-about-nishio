# 自己紹介bioの時系列分析

`自己紹介.md` は「ちょこっと自己紹介しないといけない時用」のページ。短いbioが**新しいものを上に**追記される形で蓄積されている。最古は2008年頃の著者紹介、最新は2026年（未踏ジュニア2026）。

このページ自体が**何が中心関心として残り、何が消え、何が新しく入ったか**の自然な記録になっている。

## 一貫している軸

複数のbioに繰り返し現れる固定軸：

- **「知的生産性の向上（手段）と文明の進歩（目的）」** — 2018年以降のbioで定型化、現在も使用
- **サイボウズ・ラボ 主幹研究員（2007〜）** — 全期間共通の所属
- **博士（理学）** — 経歴の起点（24歳で取得、2006年）

## 関心の重心が動いたタイミング

bioに現れる「何の研究をしているか」のフォーカスが、時期ごとに明確に動いている：

### 2008〜2013: プログラミング言語の進化
- 著書: 『Jythonプログラミング』(2008), WEB+DB『言語設計の基礎知識』(2011), 『コーディングを支える技術』(2013)
- bio中の表現: 「ソフトウェアによる知識創造の効率化、ソフトウェアの設計と進化に関心」「プログラミング言語の進化の歴史」
- セキュリティ＆プログラミングキャンプ2011で言語クラスのクラス長

### 2014〜2018: 機械学習・自然言語処理への拡張
- 著書: 『word2vecによる自然言語処理』(2014), 『エンジニアの知的生産術』(2018)
- 2014年 技術経営修士取得
- 2017年 BeProudにて機械学習関連の技術顧問（パラレルワーク開始）
- 2018年 東京工業大学 特定准教授（〜2024/3）

### 2018〜2023: 個人の知的生産技術として整理
- 『エンジニアの知的生産術』が中心著作になる
- bio中の表現: 「知的生産をエンジニア視点で分析」
- 2015年 一般社団法人未踏 理事就任、未踏ジュニア コファウンダー（2016〜）

### 2023〜2024: AI×知的生産への転換点
- 2023年 サイボウズAIエキスパートチーム兼務
- 2024年 シン東京2050プロジェクトに安野貴博氏と関わる
- 「AIによるブロードリスニング」が初登場

### 2024〜2026: Plurality・ブロードリスニングが中心へ
- 『⿻ 數位 Plurality』寄稿者・編集者、日本語化プロジェクトリーダー
- 邦訳『PLURALITY 対立を創造に変える、協働テクノロジーと民主主義の未来』（サイボウズ式ブックス）
- 都知事選2024、参院選2025、デジタル民主主義2030の広聴AI開発
- bio中で「**ブロードリスニング = 人間の他者理解能力を増強する『知的生産性向上』**」と再定義
- → 既存軸（知的生産性向上）と新しい関心（ブロードリスニング）を接続する解釈フレーム

### 2025: 退任
- 一般社団法人未踏 理事を退任（2015〜2025、10年）

## 観察される変化のパターン

1. **対象スケールの拡大**: 言語(プログラム) → 個人(エンジニア) → 組織(チーム) → 社会(民主主義/世論) → 文明
2. **「知的生産」の中身の更新**: 紙とカード（情報カード、KJ法）→ Scrapbox/外部脳 → AI支援 → 集団の他者理解
3. **手段の軸は不変、対象は拡張**: 「知的生産性の向上」というレトリックを残しながら、その射程を段階的に広げている

## 「メンテしてない」と本人が言うもの
