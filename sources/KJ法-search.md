## 観察

### nishio研究のmethodological core

KJ法は「**断片(付箋/カード/Dot)を空間配置して構造を発見する**」という手続き。
これは：
- [[../wiki/syntheses/これは何？.md|外部脳]] の基本動作（Scrapboxのリンクとカード）
- [[../wiki/topics/ブロードリスニング.md|ブロードリスニング]] のスケール拡大版（百万人の意見をクラスタリング）
- ConnectingDots（提案中、[[../wiki/meta/proposed-next-pages.md|未着手ページ案]]）
- Kozaneba（自作ツール）

…全てに通底する原型操作。**nishio の研究を貫く方法論の核** と言える。

### 60年スパンで同じ問題に取り組み続けている

- 川喜田二郎 1960年代 → nishio 2014年 word2vec → 2024年 AIでKJ法 / Audrey TangとKJ法
- 技術が変わるたびに**同じ「断片→構造発見」問題を再実装**
- これは [[../wiki/syntheses/nishio観察パターン.md#p1-固定軸--射程拡大パターン|P1 固定軸+射程拡大]] の最も鮮明な例

### Audrey TangとKJ法 = 国際展開の節目

- KJ法は元来日本の手法
- Audrey Tang（台湾）との接続は、KJ法を **国際的・集団的に展開する** 新フェーズの起点
- ブロードリスニング = KJ法の集団・統計的拡張版、と読める
