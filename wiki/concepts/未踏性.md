# 未踏性

nishio が **23年（2002〜2025）** 未踏に関わる中で継続的に理論化を試みている概念。
本人の整理は2層に分かれる：

- **個人レベル**: 個々の応募者・採択者にとっての「未踏性」
- **システムレベル**: 未踏という制度の本質（ネットワーク形成システム / プロジェクト発生成長のプロセス）

ただし本人は **「未踏とはこうである」と定義することを意図的に避けている**。PM個々人が自分の価値判断軸で判断していくことに意義があるという立場。

## 1. 個人レベルの未踏性

[未踏性とは.md](../../raw/external_brain_in_markdown/pages/未踏性とは.md) 中、本人の整理：

> **新しいものを自発的動機に基づいて作ること**

### 必要条件・否定条件

- **自発的動機（必須）**
  - 「自分の価値観に基づいて」と言い換え可
  - NG: 「研究室の先生がこれをやるといいと言ったので」「少子高齢化のためにこれが必要なので」
  - OK: 「とにかく私はこれを作りたいんです！」
- **新しさ（必須）**
  - 自発的動機だけでは不十分（「マインクラフトのクローンを作りたい」では未踏性が低い）
- **技術的高度さ（必須ではない、ベターではある）**
  - 高度さだけを追えば「先人が積み上げた山に石を積む」になる → 先人の価値観に従ってしまい、**自分の価値観が弱まる**

### 「新しさ」の自明でなさ

- 「すべての人が新しいと思うもの」は存在しない（コロンブスの卵）
- 一般大衆基準に合わせると「ちょっとだけ変えたもの」が有利になる（大衆は「速い馬が欲しい」と言う）
- → 評価軸自体が固有・分散している

## 2. システムレベルの未踏性: ネットワーク形成システム

[ネットワーク形成システムとしての未踏.md](../../raw/external_brain_in_markdown/pages/ネットワーク形成システムとしての未踏.md) — 2011年論考（情報処理学会誌, 52(12), 2011-11-15）

### 中心テーゼ

> **未踏とは技術的に高い素養を持った人を選抜し、社会関係資本を集中させるシステム**

### 構造論

3種類のコミュニティ構造：

- **左**: 密だが情報流通の範囲が狭い
- **中**: 小さな密コミュニティがつながりあうネットワーク構造 ← **理想形**
- **右**: 情報流通は広いが1対1関係のため囚人のジレンマに脆弱

→ **小規模で密なコミュニティが、両方に属する個人を介してつながる構造**が好ましい。

### 未踏の実装

- 同期の横糸（ブースト会議）と採択年度を超えた縦糸（OB-OG）が編み上げるネットワーク
- 共通の友人の存在による [[../../raw/external_brain_in_markdown/pages/囚人のジレンマ]] の破壊
- 異分野の結合 = 面白いものの源泉

### 関連する後発の整理

- 人的ネットワーク形成システム
- 直接のマッチングが人的ネットワーク形成に重要
- コミュニティによる生産性向上のすすめ

## 3. システムレベルの未踏性: プロジェクト発生成長のプロセス

[2020未踏社団:プロジェクト発生成長のプロセス.md](../../raw/external_brain_in_markdown/pages/2020未踏社団:プロジェクト発生成長のプロセス.md) — 2020〜2023に**4世代継続改訂**。

### モデル要素

```
[高度IT人材プール]
       ↓
[繰り返し会う] ← サイクル駆動
       ↓
[芽生え]: 明確な課題 × 解決アイデア × 熱意を持つ人(複数)
       ↓
[協働プロジェクト] ← コラボレーションハブの主役
       ↓
[育てる]: PDCA、リソースの接続
       ↓
[社会実装]
       ↓
[イノベーション] = 経済的価値 + 社会的価値
```

### 重要な細部

- **「ぼんやりとした課題感」→「明確な課題」**の変換に共感者・熱意を持つ人が必要（自分ごと化）
- **「熱意を持った人」が複数人にわたる時**、協働が始まる = 成功の一形態
- 関連: [集まるのが最初の一歩、一緒に居続けるのが進歩、一緒に働くのが成功](../../raw/external_brain_in_markdown/pages/集まるのが最初の一歩、一緒に居続けるのが進歩、一緒に働くのが成功.md)
- **現金は弱い資本**: 市場から調達できるリソースは稀少ではない → **調達困難リソース**が大事
- 「協働プロジェクト」と限定するのは、未踏社団の役割が**未踏人材の間の接続**だから（個人プロジェクト支援は既に社会に選択肢がある）

## 4. VC との対比（外部からのレンズ）

[未踏がVCだったら.md](../../raw/external_brain_in_markdown/pages/未踏がVCだったら.md) — 2023年3月の外部議論を本人が記録

- VCとして見れば「PFN・グノシー・落合陽一でリターン数百倍」級
- ただし契約上は **業務委託** で、運営側に金銭リターンは無い（**国の育成事業**）
- → 「投資VCに匹敵するアウトプット」を持ちながら、**インセンティブ構造はVCではない**

これは未踏性の重要な側面：**金銭リターンを目的としない選抜・育成・ネットワーク形成**。

### 改訂が滞留している理由（推測）

- 改訂のスコープが曖昧（読みやすさだけ？2020〜2023モデルとの統合？）
- 改訂の媒体が不明（学会誌再投稿？Scrapbox？単独記事？）
- 関連する整理が複数に分散（2011原稿 / 2020プロセス図 / 個別概念ページ群）
- → このWikiの本ページが**改訂版の素材を1ページに集約**している（部分的な解消）

### Revisedの叩き台として使える素材

- **原稿2011**: 構造論（密小コミュニティのネットワーク）+ ブースト会議事例
- **2020モデル**: 動学的プロセス（課題感→協働→社会実装）
- **未踏ジュニア経験**: 2020モデルの「流入」が拡張される
